Symmetry

Light & dark by rule

Dark mode is derived from light by mirroring across the palette axis — neutrals at 105−step, chromatic at 110−step. The brand even shifts blue→cyan.

Single source

Tokens compile to CSS

Primitives and semantic themes live as JSON, compile to tokens.css custom properties, and drive both the docs and your product.

Reference

Foundations

The 18-family color palette, the Open Sans type scale, spacing, radius, shadow, and icon sizes — rendered live from the tokens.

Components

Buttons, pills, selection controls, text fields, and links — each documented with states and mapped to semantic tokens.

For AI

Use it with Claude

Point Claude Code (or any LLM tool) at the repo. It reads CLAUDE.md for the rules, the token JSON for values, and the component specs for behaviour — then builds UI that matches.
